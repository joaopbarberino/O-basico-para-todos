Precisando de ajuda?

O Site
  1. Acima podemos ver a barra de navegação do site, você pode interagir clicando nas palavras.
  2. Inicio: redireciona para página inicial.
  3. Contato: área especial para você nos enviar email com uma sugestão, elogio ou outra dúvida não respondida aqui.
  4. Ajuda: redireciona para essa pagina. Use sempre que precisar de ajuda com algo.
  5. Entrar: redireciona para página de login para que você possa enviar os próprios materiais.
  6. Cadastre-se: redireciona para página de cadastro para que você possa criar seu login.
  1. Acima podemos ver a barra de navegação após o login ser realizado.
  2. "Entrar" e "Cadastre-se" dão espaço à três novas opções.
  3. "Entrar" foi substituido pelo Nome indicado no cadastro, que te redireciona para seu perfil. Lá você poderá ver TODOS os livros que possuímos em nosso banco de dados e também poderá preencher o formulário para enviar o seu próprio. Falaremos disso mais tarde.
  4. Trocar Senha: redireciona ao formulário de troca de senha. Obs.: Só é possível trocar a senha caso você saiba sua senha antiga.
  5. Sair: Encerra sua sessão. Você perderá o acesso ao seu perfil até que Entre de novo.
Consultando Livros
  1. Para consultar um livro, vá para página inicial, desça a página até a área "Sobre nós", lá você encontrará o botão abaixo:
  2. Ao clicar, o site redireciona para página de consulta:
  3. Só de chegar aqui você será capaz de ver todos os nossos livros, mas isso provavelmente não será muito útil. Por isso temos o Filtro, preencha os campos desejados e depois confirme clicando no botão "Filtrar":
    1. Nome do Livro contém: Insira o nome do livro, completo ou não, ou apenas algumas letras, o site te mostrará todos os livros que tem esse nome, ou que tenha essas letras no nome.
    2. Nome do autor contém: Funciona da mesma forma que o campo acima, mas pesquisa pelo nome do autor. Não se preocupe, com a formatação, fazemos isso por você.
    3. Gênero: Clique uma vez para ver os gêneros disponíveis e em seguida clique na opção desejada.
  4. Caso queira remover seu filtro, deixe todos os campos como na imagem acima e clique em "Filtrar".
Registrando Livros
  1. Para registrar um livro, você deve estar cadastrado e logado, você pode aprender a fazer isso na seção "O Site" no começo desta página.
  2. Acesse seu perfil.
  3. Clique em "Adicionar", isto redireciona para o formulário de cadastro de livro.
  4. Preencha o formulário seguindo as instruções e quando terminar, utilize os botões ao fim da página para decidir o que fazer em seguida. Todas as opções salvam o seu livro no nosso banco de dados e nossos moderadores darão uma olhada nele para ver se está tudo de acordo.
  5. Caso você veja uma imagem igual ou semelhante à essa, significa que tudo deu certo.
Contato
  1. Possui outra dúvida? Entre em contato conosco através do formulário em "Contato".
  2. No campo "Seu email", ponha um email válido de sua preferência, só para nós sabermos para onde enviar a resposta.
  3. No campo "Mensagem", deixe sua menssagem, caso seja uma dúvida, pedimos que você explique o máximo possível, assim poderemos te ajudar.
  4. Quando terminar, clique em enviar e aguarde a mensagem de confirmação.
  5. Pronto! Já recebemos sua menssagem!
